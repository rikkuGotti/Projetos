Tag audio

Exemplos de áudio direto de arquivos

Os exemplos abaixo usam arquivos locais. Para funcionar no navegador, coloque os arquivos dentro da pasta media do projeto com os mesmos nomes usados aqui.

Som das ondas

Exemplo com arquivo .mp3 para ambientar uma página sobre praias.

Brisa do litoral

Exemplo alternativo com arquivo .ogg, útil para compatibilidade.

Estrutura usada

A tag audio aceita atributos como controls, autoplay, loop e muted. 

<audio controls>
  <source src="media/ondas-do-mar.mp3" type="audio/mpeg">
</audio>

Tag video

Exemplos de vídeo direto de arquivos

O HTML5 também permite reproduzir vídeos locais. Os exemplos abaixo esperam arquivos dentro da pasta media.

Passeio na praia

Vídeo local em .mp4 com controles nativos do navegador.

Mergulho no mar

Exemplo com formato .webm, outra opção comum para vídeo na web.

Estrutura usada

A tag video aceita controls, poster, width, height, loop e mais. 

<video controls poster="capa.jpg">
  <source src="media/passeio-na-praia.mp4" type="video/mp4">
</video>

Vídeo incorporado

Exemplo de vídeo do YouTube

Quando o vídeo não está no projeto, podemos incorporar conteúdo externo com iframe, como no exemplo abaixo.

Vídeo do YouTube sobre praias

Quando usar cada opção

  • Use audio e video quando os arquivos estiverem no seu projeto.
  • Use iframe quando quiser incorporar vídeos hospedados no YouTube.
  • Prefira arquivos organizados em uma pasta como media.

Organização

Arquivos esperados na pasta media

media/
├─ ondas-do-mar.mp3
├─ brisa-litoral.ogg
├─ passeio-na-praia.mp4
└─ mergulho-no-mar.webm

Depois de colocar esses arquivos na pasta media, os exemplos locais passarão a funcionar automaticamente.
